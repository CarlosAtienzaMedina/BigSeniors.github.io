COLLABORATIONS

THIS PROJECT HAS BEEN ABLE TO BE CARRIED OUT, THANKS TO THE COLLABORATION OF:

CRUZ ROJA

Thanks to "Cruz Roja", we have been able to have all the necessary materials for this project, since they have donated some basic objects to us to entertain all the elderly.

GRANDES AMIGOS

Thanks to "Grandes Amigos", we have been able to plan outdoor activities, as they have helped us organize routes through the countryside and excursions to museums and monuments. And some volunteers from this great company have also helped us.

CUIDEO

Thanks to "Cuideo", we have been able to carry out and start this project thanks to the fact that we have been able to have enough money to be able to buy the necessary objects to entertain the elderly, and they also left us a building to be able to be inside it.
